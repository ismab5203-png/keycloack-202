# Lab 01 — Crear la aplicación Angular

## Paso 03 — Ejecutar y validar la aplicación base

---

## 🎯 Objetivo

Verificar que la aplicación Angular arranca correctamente con la estructura mínima preparada.

---

## 📍 Ubicación

Dentro de:

```
apps/angular-app/angular-app
```

---

## ▶️ Instalar dependencias

Si no lo has hecho todavía, ejecuta:

```bash
npm install
```

---

## ▶️ Iniciar servidor de desarrollo

Ejecuta:

```bash
npm start
```

---

## 🌐 Acceder desde el navegador

Abrir:

```
http://localhost:4200
```

---

## 🔎 Validar funcionamiento

Comprobar que:

* Se muestra el título: **Keycloak Training App**
* El menú permite navegar entre:

  * /
  * /protected
* Ambos componentes cargan correctamente.
* No existen errores en consola.

---

## 🧠 Estado actual

* La aplicación Angular funciona correctamente.
* La navegación está configurada.
* No existe autenticación todavía.
* La base está lista para integrar Keycloak en el siguiente laboratorio.
